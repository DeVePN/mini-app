# Icon Placeholder

Add your app icon here as `icon.png`

**Requirements:**
- Format: PNG
- Size: 512x512px recommended
- Background: Transparent or solid color
- Design: Simple, recognizable icon

**Telegram Mini App Icon:**
- Used in bot menu
- Shown in app sharing
- Displayed in Telegram UI

**TON Connect Manifest:**
Update the `iconUrl` in `tonconnect-manifest.json` to point to this icon.

You can generate icons using:
- Figma
- Canva
- DALL-E / Midjourney
- Icon generators online
